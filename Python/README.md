## 文档风格说明

本系列手册参考 [Go 语言学习笔记之环境配置.md](../Go/Go 语言学习笔记之环境配置.md) 的风格编写，特点包括：

- **结构清晰**: 每章有明确的目标和步骤
- **命令完整**: 所有命令可直接复制执行
- **多平台支持**: Windows/macOS/Linux 命令分别标注
- **表格辅助**: 使用表格对比选项和快捷键
- **问题排查**: 每章包含常见问题及解决方案
